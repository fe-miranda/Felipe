# 🌈 Mundo das Descobertas

Jogo educativo mobile para crianças a partir de 3 anos, feito do zero com **Expo + React Native + TypeScript**.

## ✅ O que o jogo tem

- 🎨 Jogo de cores (toque na cor correta)
- 🐾 Jogo de bichinhos (identificação com feedback sonoro em texto)
- 🔷 Jogo de formas (reconhecimento visual)
- 🚫 Sem anúncios
- 👶 UI com botões grandes e linguagem simples

## ▶️ Rodar localmente

```bash
cd MundoDescobertas
npm install
npm start
```

## 🧪 Validar projeto

```bash
cd MundoDescobertas
npm test
```

## 📦 Baixar APK pelo GitHub Actions (Artifacts)

1. Abra a aba **Actions** no GitHub.
2. Execute o workflow **Build APK — Mundo das Descobertas** (ou use um run automático do push).
3. Ao finalizar, vá em **Artifacts**.
4. Baixe o arquivo `mundo-das-descobertas-v<run_number>.apk`.
